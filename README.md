# PlantApp_UIKit

A CRUD app that allow users to create and save a collection of plants with specified name, picture, water date, and watering habit.

Features include:
- Allowing users to choose photos for plant through Photo library or Camera.

- Push Notification to remind users to water plant.

- Allowing users to mark plant as watered through notification banner through UNNotificationAction.

- Offline data persistence through Core Data.

- Cloud data persistence through Google Firebase.

- Written in Swift
